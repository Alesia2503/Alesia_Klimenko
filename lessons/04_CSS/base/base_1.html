Форматы файлов.

.DOC или .doc (от англ. document) - расширение имени файла, используемое для файлов, представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения простых текстовых файловбез без формирования, однако позже стало использоваться для двоичных форматов с разметкой.

Portable Document Format (PDF ) - межплатформенный формат электронных документов, разработанный фирмой Adobe Systems с использованием ряда возможностей языка
Postscript.
Документ PDF
ZIP - популярный формат сжфтия данных и архивации файлов. Файл в этом формате обычно имеет расширение .zip и хранит в сжатом или несжатом виде один или несколько файлов, которое можно из него извлечь путем распоковки с помощью специальной программы.
